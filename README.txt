WEB DE FIRMAS - MAESTRO SURQUILLO

Archivos:
- index.html: página pública para ingresar DNI, nombre y firma.
- admin.html: panel de aprobación.
- styles.css / script.js: diseño y funciones.
- maestro-surquillo.png: imagen principal.

Clave demo del panel:
maestro22

Importante:
Esta versión es una maqueta/prototipo. Guarda datos en el navegador con localStorage.
Para usar DNI reales en internet necesitas una versión segura con servidor, base de datos,
HTTPS, aviso de privacidad y control de acceso real.
